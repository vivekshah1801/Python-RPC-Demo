# Python RPC Demo
## Demonstrating Remote procedure calls over RPC using RpyC
<hr>

# Installation

#### On server
- `pip3 install -r requirements.txt`
- `python3 server.py`

#### On Client
- `pip3 install -r requirements.txt`
- `python3 client.py`
